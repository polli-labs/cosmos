# Security Policy

## Supported Versions

Security fixes are provided for the latest release line.

| Version | Supported |
| --- | --- |
| 0.6.x | yes |
| <= 0.5.x | no |

## Reporting a Vulnerability

Please do not report security issues in public GitHub issues.

Preferred path:

1. Open a private GitHub Security Advisory for this repository:
   https://github.com/polli-labs/cosmos/security/advisories/new
2. Include:
   - affected version(s)
   - reproduction steps or proof of concept
   - impact assessment
   - suggested mitigation (if known)

If private advisory access is unavailable, open a normal issue with minimal details and ask maintainers for a secure channel before sharing exploit specifics.

## Disclosure Process

- We aim to acknowledge reports quickly and provide a triage update within 5 business days.
- Valid reports will be patched in the next available release lane, with credit if requested.
